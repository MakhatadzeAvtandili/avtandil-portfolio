# Aurora Glow Pricing Table v2.0.0

Thank you for purchasing the Aurora Glow Pricing Table! This is a premium, fully responsive pricing table component designed to give your website a standout, modern look.

Version 2.0 is a complete visual overhaul focused on dynamism, interaction, and creating a stunning user experience.

## Key Features

  • **Framework-Free:** Built with pure HTML, CSS, and JavaScript. No dependencies or bloat.
  • **Dynamic "Aurora" Background:** A live, animated gradient background that gives the design depth and a unique feel.
  • **Glassmorphism Effect:** Frosted-glass style cards that react to the background, creating a premium aesthetic.
  • **3D Hover Animations:** Cards gain a subtle 3D perspective on hover, making the experience more interactive and engaging.
  • **Fully Responsive:** Looks perfect on all devices, from mobile phones to 4K desktops.
  • **Easy Customization:** The design can be easily themed by changing a few CSS variables.
  • **Accessibility Ready:** Built with WCAG 2.1 AA standards in mind, including keyboard navigation and high-contrast text.
  • **Clean Code:** Well-commented and structured with BEM methodology for easy maintenance.

## Quick Start (Installation)

  1. Copy Files: Copy the assets folder into your project's directory.
  2. Link CSS: In the <head> of your HTML file, add the link to pricing-table.css. 
    ```html
<link rel="stylesheet" href="path/to/assets/css/pricing-table.css">
```
  3. Link JavaScript: Before the closing </body> tag, add the link to pricing-table.js. 
    ```html
<script src="path/to/assets/js/pricing-table.js"></script>
```
  4. Add HTML Structure: Copy the HTML code from index.html. Important: The structure must be placed inside the <main class="pt-container"> wrapper, which holds both the aurora background elements and the pricing table itself. 
    ```html
<main class="pt-container">
    <div class="aurora-bg aurora-bg--1"></div>
    <div class="aurora-bg aurora-bg--2"></div>
    <div class="aurora-bg aurora-bg--3"></div>

    <section class="pt" id="pricing">
            </section>
</main>
```

# Customization & Theming (v2.0)

The Aurora Glow Pricing Table is designed for easy and flexible customization. You can change the core aspects of the design by editing just a few lines of CSS.

## Customizing the Aurora Background

The most unique part of the design, the dynamic background, is controlled by the .aurora-bg classes.

To change the background colors, open pricing-table.css and find the following selectors:

  • .aurora-bg--1
  • .aurora-bg--2
  • .aurora-bg--3

In each of these, you can change the background-color property to any color you like.

Example: Creating a "Sunset" effect:

```css
.aurora-bg--1 {
  background-color: #ff5e00; /* Orange */
  /* ... */
}
.aurora-bg--2 {
  background-color: #ff007f; /* Pink */
  /* ... */
}
.aurora-bg--3 {
  background-color: #6a00ff; /* Purple */
  /* ... */
}
```

Note: For advanced customization, you can edit the @keyframes aurora-move rules to change the animation speed and movement path.

Core CSS Variables
Global design parameters are controlled by CSS variables located at the top of pricing-table.css inside the :root selector.

Variable   Description
Colors   
--pt-text-primary   Main text color (headings, prices).
--pt-text-secondary   Secondary text color (descriptions, period).
--pt-accent-start   The starting color of the main gradient (buttons, icons).
--pt-accent-end   The ending color of the main gradient.
--pt-card-bg   The card background color (use rgba for transparency).
--pt-card-border-color   The card border color.
Sizing & Spacing   
--pt-max-width   The maximum width of the component.
--pt-gap   The space between the pricing cards.
--pt-card-padding   The inner padding within each card.
--pt-card-radius   The border-radius for the cards.
--pt-btn-radius   The border-radius for the buttons.
Animation   
--pt-speed   The base speed for all animations.
--pt-easing   The timing function for animations (e.g., cubic-bezier).

Adding or Removing a Plan
To Add: To add a new plan, simply copy an entire <article class="pt__card">...</article> block and paste it inside the <div class="pt__cards"> container. The grid system will automatically adjust the layout.

To Remove: To remove a plan, just delete its corresponding <article> block.

პროფესიონალური პორტფოლიოს ვებ-გვერდი

პროექტის მიმოხილვა

ეს პროექტი წარმოადგენს თანამედროვე, სრულად საპასუხო (responsive) და ორენოვან (ქართული/ინგლისური) პორტფოლიოს ვებ-გვერდს, რომელიც შექმნილია უახლესი ვებ-ტექნოლოგიების გამოყენებით. მისი მიზანია დეველოპერისა და დიზაინერის უნარებისა და ნამუშევრების პროფესიონალურად წარმოჩენა.

გამოყენებული ტექნოლოგიები

პროექტის ასაწყობად გამოყენებულია შემდეგი ძირითადი ტექნოლოგიები და ბიბლიოთეკები:

  • React + Vite: სწრაფი და თანამედროვე JavaScript ბიბლიოთეკა UI-ის ასაწყობად და Build Tool.
  • Tailwind CSS v4: Utility-first CSS ფრეიმვორკი სწრაფი და მოქნილი სტილიზაციისთვის (CSS-first კონფიგურაცია).
  • React Router: გვერდებს შორის ნავიგაციისა და მარშრუტიზაციის სამართავად.
  • react-i18next (i18next): საიტის ორენოვანი ფუნქციონალის უზრუნველსაყოფად (EN/KA) და თარგმანების სამართავად.
  • Framer Motion: დახვეწილი და ეფექტური ანიმაციების შესაქმნელად.
  • EmailJS: საკონტაქტო ფორმიდან შეტყობინებების პირდაპირ იმეილზე გასაგზავნად, ბექენდის გარეშე.
  • react-helmet-async: SEO-სთვის მნიშვნელოვანი მეტა-თეგების (სათაური, აღწერა) დინამიური მართვისთვის თითოეული გვერდისთვის.

ძირითადი ფუნქციები

  • ორენოვნება: მარტივი გადართვა ქართულ და ინგლისურ ენებს შორის (ნაგულისხმევი: ქართული). ენის არჩევანი ინახება localStorage-ში.
  • საპასუხო დიზაინი: საიტი იდეალურად გამოიყურება ყველა ზომის ეკრანზე (320px-დან ზემოთ).
  • სექციები: მთავარი გვერდი დაყოფილია ლოგიკურ სექციებად (Hero, Workflow, Projects, Skills, Testimonials, Contact).
  • ბლოგი: დინამიური ბლოგის სექცია პოსტების სიით და ცალკეული პოსტის გვერდებით (/blog/:slug).
  • SEO ოპტიმიზაცია: თითოეულ გვერდს აქვს უნიკალური სათაური და აღწერა უკეთესი საძიებო სისტემების შედეგებისთვის.
  • ჩატვირთვის ანიმაცია (Preloader): უნიკალური და პროფესიონალური ანიმაცია საიტის პირველადი ჩატვირთვისას.
  • საკონტაქტო ფორმა: EmailJS-ის ინტეგრაცია ფორმის მონაცემების პირდაპირ იმეილზე მისაღებად.
  • Testimonials: კლიენტების რეკომენდაციების დინამიური სლაიდერი.
  • თანამედროვე დიზაინი: "Summer Breeze" კონცეფცია ღია ფერებით, რბილი ანიმაციებით და ჰაეროვანი განლაგებით.
